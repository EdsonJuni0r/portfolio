# 🌐 Portfólio — Edson Junior | Pedreno Tech & Infra

Portfolio pessoal em **React + TypeScript + Vite**, hospedado no GitHub Pages.

## 🚀 Deploy (GitHub Actions — automático)

1. Crie o repositório: `EdsonJuni0r/portfolio`
2. Envie o código:
   ```bash
   git init && git add .
   git commit -m "feat: portfolio inicial"
   git branch -M main
   git remote add origin https://github.com/EdsonJuni0r/portfolio.git
   git push -u origin main
   ```
3. **Settings → Pages → Source → GitHub Actions**

🔗 URL: `https://EdsonJuni0r.github.io/portfolio/`

## 🛠️ Local

```bash
npm install && npm run dev
# http://localhost:5173/portfolio/
```
